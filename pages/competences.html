Les compétences acquises en première année de BTS SIO option SLAM.

01

Soutenir et conseiller les utilisateurs

Support, assistance et conseil aux utilisateurs du système d'information

Support utilisateur Assistance technique Résolution d'incidents Diagnostic réseau Documentation technique Windows Server 2025 Active Directory

Dans le cadre du projet Les Forgeurs de Pixel (avec Paul Faudois, décembre 2025), j'ai participé à la mise en place et à la documentation complète d'une infrastructure réseau et serveur sous Windows Server 2025. Ce projet consistait à répondre à un appel d'offre en proposant une solution informatique complète pour une entreprise fictive.

J'ai contribué à la création et à la gestion des comptes utilisateurs avec des droits adaptés (administrateur, utilisateur standard), à la mise en place du service de fichiers avec des permissions NTFS par rôle, ainsi qu'à la configuration du service RDP pour l'accès distant. J'ai également rédigé la documentation réseau et serveur, incluant le plan d'adressage IP (VLANs), les schémas réseau physique et logique, et la liste des vulnérabilités connues (CVE) sur Windows Server 2025.

02

Développer des solutions applicatives

Conception, développement et maintenance de solutions logicielles

HTML5 / CSS3 JavaScript PHP (MVC) SQL / SQL Server Git / GitHub / GitLab VS Code WordPress (CMS) IIS (serveur web) FTP / FileZilla

Au cours de la première année, j'ai développé plusieurs compétences en développement applicatif à travers des ateliers professionnels :

Projet Boutique Pic du Midi (AP2.1 – M1 – E1) : création d'une page statique d'accueil avec le CMS WordPress (thème North Shore), intégrant la charte graphique du client (logo, bannière, image principale, menu de navigation). Ce projet m'a permis de maîtriser la personnalisation d'un thème, la gestion des médias et la publication de pages statiques sous WordPress.

Projet NetcarMVC — Gestion de versions avec Git (AP2.2 – M0) : découverte et mise en pratique du logiciel de gestion de versions Git dans VS Code. J'ai appris à initialiser un dépôt, suivre les modifications, réaliser des commits avec messages, et pousser un projet vers un dépôt distant GitLab. Ce projet m'a initié aux bonnes pratiques du développement collaboratif.

Projet Les Forgeurs de Pixel : configuration et déploiement d'un serveur IIS (Internet Information Services) sous Windows Server 2025, publication de sites web personnels, mise en place d'un service FTP (avec FileZilla), et création d'une base de données SQL Server avec les tables jeux et developpeurs.

03

Gérer le patrimoine informatique

Sécurité, maintenance et gestion du système d'information

Sécurité des données Gestion des droits NTFS VLAN / Plan d'adressage IP Pare-feu (Zyxel) Veille CVE / CERT-FR Windows Server 2025 RDP Devis matériel

Dans le cadre du Projet Les Forgeurs de Pixel, j'ai participé à la gestion du patrimoine informatique d'une infrastructure simulée. Cela comprenait :

La conception d'un plan d'adressage IP structuré en VLANs (Management, Direction, Créatif, Wi-Fi, Serveurs, DMZ) avec un pare-feu Zyxel assurant la sécurité périmétriche.

La rédaction d'un devis matériel complet (PC portables Dell XPS 13, serveur PowerEdge R7615, switchs Zyxel, pare-feu) pour un total de 43 436,64 € TTC, répondant aux besoins de l'entreprise cliente.

La réalisation d'une veille sur les vulnérabilités de Windows Server 2025, avec identification et documentation des CVE ouvertes depuis les bulletins Microsoft et le CERT-FR.

La mise en place d'une politique de droits sur les dossiers partagés (Documents, Technique, Gouvernance) avec des permissions adaptées à chaque rôle utilisateur.
